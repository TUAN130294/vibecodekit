# Tech Stack

## Frontend (optional)
- React or Vue with TypeScript (for dashboards).

## Backend
- Node.js (Express/Fastify) with TypeScript strict mode.
- Python (FastAPI for services, scripts for ETL/ML).

## Python Tooling
- Celery for background tasks; pandas/numpy for transforms.
- scikit-learn/OpenCV/YOLO for ML/CV tasks.

## Data & Infra
- PostgreSQL 14+ as primary DB.
- Redis for caching and queues (Bull Queue).
- Elasticsearch optional for search/analytics.
- RabbitMQ or BullMQ for messaging/queues.

## Automation
- n8n self-hosted for orchestration.
- Google Sheets API for ops workflows.

## Cloud & Delivery
- Cloudflare (CDN/Workers) at the edge; AWS/GCP for core services.
- Docker containers; Kubernetes optional for scaling.
- GitHub Actions for CI/CD.

## Integrations & APIs
- Google Sheets API, Stripe, SendGrid, WhatsApp Business API.

## Monitoring & Observability
- Datadog or ELK stack; structured logging everywhere.

## Testing
- Jest (Node), pytest (Python), Playwright for E2E.

## Package Managers
- npm (Node), pip (Python).
